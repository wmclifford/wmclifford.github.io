# Privacy Policy

**Last updated:** June 17, 2026

This privacy policy describes how LUMIS ("the application") handles data accessed through Google Workspace APIs.

## What Data Is Accessed

LUMIS accesses the following data from your Google account:

- **Gmail (read-only):** Email message metadata (sender, subject, date, labels) and message content for the purpose of surfacing unread and important messages. LUMIS does not send, modify, or delete emails.
- **Google Calendar (read and write):** Calendar event details (title, date, time, description, attendees) for the purpose of viewing your schedule and creating new events.

## How Data Is Used

All data retrieved from Google APIs is:

- **Processed locally** on the machine where LUMIS is installed
- Used solely to provide the features described on the [home page](/)
- Never sold, shared with third parties, or used for advertising
- Never transmitted to external servers or cloud services beyond the Google APIs themselves

## Data Storage

- Data retrieved from Google APIs is stored locally on your machine in a private directory
- No data is stored in cloud databases, remote servers, or third-party services
- Cached data can be deleted at any time by removing the local data directory

## Data Retention

LUMIS retains locally cached data only as long as needed for its features to function. You may delete all locally stored data at any time.

## Third-Party Disclosure

LUMIS does not share, sell, or disclose any user data to third parties. No analytics, tracking, or advertising services are used.

## Google API Services User Data Policy

LUMIS's use and transfer of information received from Google APIs adheres to the [Google API Services User Data Policy](https://developers.google.com/terms/api-services-user-data-policy), including the Limited Use requirements.

## Security

Access to Google APIs is authenticated via OAuth 2.0. Credentials are stored locally and are not accessible to other applications or users on the system.

## Changes to This Policy

This policy may be updated to reflect changes in the application's functionality. The "last updated" date at the top of this page will be revised accordingly.
